# Roman Numbers
<font size="3">
<b>Roman numerals are represented by seven different symbols: I, V, X, L, C, D and M.
</b>
</font>
## Problem Description
>Given an integer x, return true if x is palindrome integer.

## Special Cases
<p align = "center">
Input: x = -121
<br>
Output: false
<br>
<br>
0 is a Palindrome Number
</p>
