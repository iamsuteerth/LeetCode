# Insert Search Position
## Problem Description
<font size="3.5">
Given a sorted array of distinct integers and a target value, return the index if the target is found. If not, return the index where it would be if it were inserted in order.
</font>

## Sample Inputs
<p align = "center">
<font size="4"><b><font color = "red">Input:</font> nums = [1,3,5,6], target = 5<br>
<font color = "lime">Output</font> = 2
<br>
<br>
<font color = "red">Input:</font> nums = [1,3,5,6], target = 2<br>
<font color = "lime">Output</font> = 1
<br>
<br>
<font color = "red">Input:</font> nums = [1,3,5,6], target = 7<br>
<font color = "lime">Output</font> = 4
</font>
